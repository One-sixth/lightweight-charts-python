# Example 32: HTMLChart_BN 功能演示

## 功能展示

本示例展示 `HTMLChart_BN` 类的完整功能，这是一个专为回测系统设计的增强版 HTML 图表。

### 主要特性

1. **多策略切换** - 左侧边栏可切换不同策略的 K 线图
2. **交易明细展示** - 表格显示开仓/平仓详情，支持双击跳转
3. **绩效指标面板** - 显示夏普比率、最大回撤、胜率等关键指标
4. **策略参数展示** - 显示每个策略的配置参数
5. **K 线图交互** - 缩放、十字光标、绘图工具等完整功能

## 运行方法

```bash
python htmlchart_bn_demo.py
```

## 输出文件

运行后会生成 `htmlchart_bn_demo.html` 文件，在浏览器中打开即可查看效果。

## 数据说明

示例使用模拟数据：
- 100 天的 OHLCV 数据
- 2 个策略（均线交叉策略、布林带策略）
- 每个策略包含 5-15 笔交易记录
- 绩效指标和策略参数随机生成
